# Sections des Références Juridiques

Chaque fichier de référence dans ce dossier suit une structure standardisée.

## Sections Obligatoires

| Section | Description |
|---------|-------------|
| **Résumé** | Description du domaine et importance pour KBouffe |
| **Textes de Loi Applicables** | Références précises aux lois, décrets, règlements |
| **Obligations pour la Plateforme** | Liste numérotée des obligations concrètes |
| **Exemple Non-Conforme** | ❌ Situation typique de non-conformité |
| **Exemple Conforme** | ✅ Bonne pratique à adopter |
| **Sanctions** | Peines et amendes en cas de violation |
| **Recommandations Techniques** | Actions d'implémentation pour KBouffe |

## Sections Optionnelles

| Section | Description |
|---------|-------------|
| **Points d'Attention** | Spécificités du contexte camerounais |
| **Jurisprudence** | Décisions de justice pertinentes |
| **Évolutions Attendues** | Projets de loi ou réformes en cours |
| **Références** | Liens vers textes officiels |

## Nommage des Fichiers

Format : `{domaine}-{sous-domaine}.md`

Préfixes par domaine :
- `fiscalite-` : Impôts, taxes, TVA
- `commerce-` : Registre de commerce, licences
- `sante-` : Hygiène, sécurité alimentaire
- `donnees-` : Protection des données personnelles
- `paiement-` : Paiements mobiles, fintech
- `travail-` : Droit du travail, emploi
- `marketing-` : Publicité, SMS, communication
- `consommateur-` : Droits des consommateurs
- `fintech-` : Obligations comptables, reporting
- `ecommerce-` : Commerce électronique, plateformes
