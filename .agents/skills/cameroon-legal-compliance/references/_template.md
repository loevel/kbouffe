# Template de Référence Juridique

Utilisez ce template pour créer de nouvelles fiches de référence juridique.

## Format

```markdown
# [Titre de la Référence]

## Résumé

Brève description du domaine juridique et de son importance pour KBouffe.

## Textes de Loi Applicables

- **[Nom de la loi]** — [Référence précise, articles]
- ...

## Obligations pour la Plateforme

1. [Obligation concrète]
2. ...

## ❌ Exemple Non-Conforme

Description d'une situation de non-conformité typique dans le contexte de KBouffe.

```
[Code ou configuration non-conforme]
```

**Risque** : [Explication du risque juridique]

## ✅ Exemple Conforme

Description de la bonne pratique à adopter.

```
[Code ou configuration conforme]
```

**Pourquoi** : [Explication de la conformité]

## Sanctions

- [Sanction 1 : type, montant, référence]
- ...

## Recommandations Techniques pour KBouffe

1. [Recommandation concrète d'implémentation]
2. ...

## Points d'Attention

- [Point spécifique au contexte camerounais]
- ...

## Références

- [Lien ou référence vers le texte officiel]
- ...
```

## Conventions

- Les titres de section sont en français
- Les exemples de code sont en anglais (comme le codebase)
- Les montants sont en FCFA (XAF)
- Toujours citer les articles de loi précis
- Inclure des recommandations techniques actionables
